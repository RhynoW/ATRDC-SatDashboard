# user_defined_TLE — 使用者自訂 TLE

把 `*.tle` 或 `*.txt` 檔放進本目錄即自動讀取顯示（索引每 10 分鐘重建，
或呼叫 `POST /api/admin/reload_cats` 立即生效）。

## 格式

- 支援 **2 行**與 **3 行** TLE，同一檔可混用
- 3 行格式的名稱行可帶 3LE 的「`0 `」前綴，名稱可含繁體中文
- `#` 開頭為註解行，空行忽略
- 編碼：UTF-8（可含 BOM）

```
# 3 行格式
衛星中文名稱
1 99001U 17049A   26182.53517132  .00000156  00000-0  50457-4 0  9993
2 99001  98.2252 257.5261 0012213  44.2454 315.9716 14.50903884468753

# 2 行格式（名稱由資料庫或衛星目錄補）
1 42920U 17049A   ...
2 42920  98.2252 ...
```

## 行為

- NORAD ID 已存在於資料庫 → **覆蓋**該衛星的 TLE（與名稱，若有名稱行）
- NORAD ID 不存在 → **新增**衛星（中繼資料可由 `user_defined_SaTCatalogue/` 補）
- 多檔案間同一 NORAD ID 重複時，檔名排序靠後者覆蓋靠前者
- API：`GET /api/user/tles` 查看解析結果與即時位置
